| 名称   | 示例   | 说明                                      |
| ------ | ------ | ----------------------------------------- |
| String | "你好" | 文字，或字符。需要英文引号。              |
| Int    | 15     | 一个数字，不需要引号。                    |
| Float  | 1.2    | 一个小数，不需要引号                      |
| Bool   | true   | 仅支持「true」或者「false」，不需要引号。 |
